# dolphin-filtering

Long-wavelength high-pass and Goldstein filtering for dolphinRust.

Part of [dolphinRust](https://github.com/morton-analytics-llc/dolphinRust) — an optimized Rust rebuild of the OPERA/DISP-S1 InSAR displacement pipeline. See the [workspace README](../../README.md) for the overview, quickstart, and validation status.
